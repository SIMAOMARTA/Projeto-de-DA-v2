Detailed Description

Classe utilitária para leitura e parsing dos ficheiros de entrada.

Fornece dois métodos públicos estáticos:

  • parseRanges() : lê o ficheiro de live ranges.
  • parseRegisters() : lê o ficheiro de configuração.

Linhas começadas por '#' ou vazias são ignoradas em ambos os ficheiros. Erros de formato lançam std::invalid_argument; ficheiros inacessíveis lançam std::runtime_error.

Faz o parse de um token individual de ponto de programa.

Parameters
tokenToken a parsear (espaços são ignorados).
Returns
ProgramPoint construído a partir do token.

Percorre o token carácter a carácter:

  • Dígitos são acumulados na string digits.
  • '+' ativa isStart.
  • '-' ativa isEnd.
  • Outros caracteres são ignorados silenciosamente.

Se não for encontrado nenhum dígito, é lançada uma exceção. A conversão para int é feita com std::stoi, que lança std::invalid_argument ou std::out_of_range em caso de erro.

std::vector< LiveRange > Parser::parseRanges ( const std::string & filename)
static

Lê e faz o parse do ficheiro de live ranges.

Parameters
filenameCaminho para o ficheiro de live ranges.
Returns
Vetor de LiveRange (um elemento por linha de dados).

Algoritmo:

  1. Ignora linhas de comentário e vazias via isCommentOrEmpty().
  2. Divide a linha pelo primeiro ':' para obter varName e a lista de pontos.
  3. Divide a lista de pontos por ',' e parseia cada token com parsePoint().
  4. Cria um LiveRange e adiciona ao vetor de resultado.

Cada linha do ficheiro gera exatamente um LiveRange, mesmo que a mesma variável apareça em múltiplas linhas. A fusão em webs é da responsabilidade do InterferenceGraph.

Parameters
filenameCaminho para o ficheiro de live ranges.
Returns
Vetor de LiveRange (um elemento por linha de dados).
Exceptions
std::runtime_errorSe o ficheiro não puder ser aberto.
std::invalid_argumentSe uma linha não contiver ':', se o nome da variável estiver vazio, ou se não houver nenhum ponto de programa válido na linha.
Complexidade
O(L × P) onde L = número de linhas do ficheiro, P = pontos por linha.

AlgorithmConfig Parser::parseRegisters ( const std::string & filename)
static

Lê e faz o parse do ficheiro de configuração de registos.

Parameters
filenameCaminho para o ficheiro de configuração.
Returns
Estrutura AlgorithmConfig preenchida.

Algoritmo:

  1. Ignora comentários e linhas vazias via isCommentOrEmpty().
  2. Divide pelo primeiro ':' para obter chave e valor.
  3. Para a chave "registers": converte o valor para inteiro.
  4. Para a chave "algorithm": verifica se há vírgula no valor:
    • Sem vírgula: algoritmo sem parâmetro (basic ou free).
    • Com vírgula: algoritmo com parâmetro K (spilling ou splitting).
  5. Valida que o algoritmo é um dos quatro suportados.
  6. Após ler todas as linhas, valida que numRegisters > 0.

Chaves desconhecidas são ignoradas silenciosamente para tolerar ficheiros com campos extra (extensibilidade futura).

Parameters
filenameCaminho para o ficheiro de configuração.
Returns
Estrutura AlgorithmConfig preenchida.
Exceptions
std::runtime_errorSe o ficheiro não puder ser aberto.
std::invalid_argumentSe uma linha não contiver ':', se o valor de registers estiver vazio ou for <= 0, se o parâmetro após a vírgula estiver vazio, ou se o algoritmo for desconhecido.
Complexidade
O(L) onde L = número de linhas do ficheiro.
