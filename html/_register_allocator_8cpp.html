Chaitin spill heuristic: pick node that maximises degree / liveRangeSize.

A web that spans many lines is relatively cheap to spill; a web with many neighbours but a short live range is expensive to keep in a register. We spill the node with the highest degree-to-size ratio.

@complexity O(|active|)
