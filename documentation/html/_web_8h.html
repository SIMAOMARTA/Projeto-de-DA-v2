Detailed Description

Declaração da classe Web.

Uma web é a união de uma ou mais LiveRanges da mesma variável cujos intervalos de programa se sobrepõem. É o nó fundamental do grafo de interferência: cada web corresponde a um valor distinto da variável que pode ser atribuído a um registo físico independente.

A construção de webs segue um algoritmo greedy de fusão descrito em InterferenceGraph::buildWebs(): parte de uma web por live range e funde iterativamente pares de webs da mesma variável que se sobreponham, até nenhum par se sobrepor. O resultado é o conjunto minimal de webs tal que cada web é uma live range maximal conexa.

Internamente, a Web mantém:

  • ranges_ : as live ranges originais fundidas nesta web.
  • lineSet_ : união dos lineSets de todas as ranges (para pesquisas rápidas).
  • annotationMap_ : cache de anotações (+/-) por linha, usada em interferesWith().
