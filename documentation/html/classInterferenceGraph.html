Detailed Description

Grafo de interferência entre webs de variáveis.

Recebe um conjunto de live ranges, funde-as em webs por variável e constrói as arestas de interferência entre pares de webs de variáveis distintas (ou derivadas) que coexistam em pelo menos um ponto de programa.

A classe é usada de duas formas:

  1. Construída a partir de live ranges (construtor principal).
  2. Copiada pelo RegisterAllocator quando precisa de preservar o grafo original enquanto experimenta modificações (spilling/splitting).

◆ InterferenceGraph() [1/2]

InterferenceGraph::InterferenceGraph ( const std::vector< LiveRange > &  ranges)
explicit

Constrói o grafo de interferência a partir de um conjunto de live ranges.

Executa duas fases:

  1. buildWebs(): agrupa as live ranges por variável e funde as que se sobrepõem (partilham pelo menos uma linha), usando um algoritmo greedy de fusão iterativa. Adiciona cada web final como vértice do baseGraph_.
  2. buildEdges(): para cada par de webs distintas, invoca Web::interferesWith() e, se verdadeiro, acrescenta uma aresta bidirecional ao baseGraph_.
Parameters
rangesLista de live ranges de entrada (pode conter múltiplas ranges da mesma variável; a fusão é feita internamente).
Complexidade
O(W² × L), onde W = número de webs resultante, L = número médio de linhas por web. Dominado pela fase buildEdges() que testa todos os pares (W² chamadas a interferesWith()).

Constrói as arestas de interferência entre todas as webs.

Testa todos os pares (i, j) com i < j. Para cada par em que Web::interferesWith() retorna true, invoca addEdge(i, j).

Complexidade
O(W² × L), onde W = número de webs, L = linhas médias por web.

◆ buildWebs()

void InterferenceGraph::buildWebs ( const std::vector< LiveRange > &  ranges)
private

Constrói as webs a partir das live ranges e adiciona os vértices ao grafo.

Algoritmo:

  1. Agrupa as live ranges por nome de variável.
  2. Para cada variável, cria uma Web por range (webs parciais).
  3. Itera até convergência: se dois parciais se sobrepõem (overlaps()), funde o segundo no primeiro (absorb()) e remove o segundo.
  4. Renumera os ids finais (0..W-1) e adiciona cada web como vértice em baseGraph_.
Parameters
rangesLive ranges de entrada.
Complexidade
O(R² × P) no pior caso, onde R = número de ranges da variável com mais ranges, P = número de pontos por range. Na prática tende a ser linear no número de ranges.

Grafo de inteiros do professor que representa a estrutura de adjacência.
