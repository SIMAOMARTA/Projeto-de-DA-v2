Detailed Description

Representa um ponto de programa (instrução) dentro de uma live range.

Cada ponto contém o número da linha de instrução e duas flags booleanas que anotam eventos de definição (isStart / '+') e de último uso (isEnd / '-') nessa instrução.

Note
O mesmo ponto pode ter ambas as flags ativas simultaneamente, como acontece em instruções do tipo "i = i + 1", onde a variável i é lida (último uso da live range anterior) e definida (início da nova live range) na mesma instrução.
Exemplo de uso
ProgramPoint def(5, true, false); // linha 5: definição de variável
ProgramPoint use(9, false, true); // linha 9: última utilização
ProgramPoint both(7, true, true); // linha 7: def. e último uso

bool ProgramPoint::isEnd = false

Indica que esta é a última utilização da variável nesta instrução.

Corresponde ao sufixo '-' no ficheiro de live ranges. Quando true, esta instrução é o ponto de fim da live range — o registo que contém o valor pode ser libertado imediatamente após esta instrução ser executada.

◆ isStart

bool ProgramPoint::isStart = false

Indica que a variável é definida (escrita) nesta instrução.

Corresponde ao sufixo '+' no ficheiro de live ranges. Quando true, esta instrução é o ponto de início de uma nova live range para a variável — o compilador gera aqui uma escrita para o registo ou posição de memória associada.
