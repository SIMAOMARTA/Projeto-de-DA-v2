Detailed Description

Configuração do algoritmo de alocação de registos.

Esta estrutura é preenchida pelo Parser a partir do ficheiro de configuração de registos e é passada ao RegisterAllocator para selecionar o algoritmo a executar e os respetivos parâmetros.

O fluxo de utilização típico é:

AlgorithmConfig cfg = Parser::parseRegisters("registers.txt");
auto [allocs, graph, meta] = RegisterAllocator::allocate(ig, cfg);
